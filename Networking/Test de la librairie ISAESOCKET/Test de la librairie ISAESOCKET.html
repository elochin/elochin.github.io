Test de la librairie ISAESOCKET

La librairie ISAESOCKET qui vous est proposée vous affranchira de quelques petits soucis de mise en forme programmatique. Vous avez le choix, ou pas, de l'utiliser pour effectuer vos programmes. Elle vous aidera à démarrer mais n'est pas exhaustive. Elle est fournie avec un code exemple de client/serveur Echo. Avant de vous lancez dans les LABs, je vous conseille de compiler et tester ce code. N'hésitez pas à le modifier pour provoquer des erreurs, c'est le meilleur moyen de se l'approprier. Enfin, modifier le code du client et du serveur Echo en utilisant les bonnes primitives de l'API socket associées à l'ouverture d'une socket SOCK_STREAM.
