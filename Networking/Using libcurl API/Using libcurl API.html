Using libcurl API

In order to brush up your C programming skills, although some of you are already experienced or wrote tons of lines during their c√©sure ;) I propose to illustrate the libcurl capabilities by writing a simple C code. The objective is to fetch a content provided by a webpage via an HTTP request. This library is well documented and avalaible here : https://curl.haxx.se/libcurl/c/

You need to link the library using -lcurl when compiling with gcc.

#include <stdio.h>
#include <curl/curl.h>

int main(void)
{
  CURL *curl;
  CURLcode res;
  FILE* file = fopen("webpage.html", "w");

  curl_global_init(CURL_GLOBAL_DEFAULT);

  curl = curl_easy_init();
  if(curl) {
    curl_easy_setopt(curl, CURLOPT_URL, "https://www.isae-supaero.fr/fr");
    /* Write the downloaded webpage in a file */
    curl_easy_setopt(curl, CURLOPT_WRITEDATA, file);
    /* Perform the request, res will get the return code !!must be added after curl_easy_setopt()!! */
    res = curl_easy_perform(curl);
    /* Check for errors */
    if(res != CURLE_OK)
      fprintf(stderr, "curl_easy_perform() failed: %s\n",
              curl_easy_strerror(res));


    /* always cleanup */
    curl_easy_cleanup(curl);
  }

  curl_global_cleanup();
  fflush(file);
  fclose(file);

  return 0;
}
