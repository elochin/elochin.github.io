Connaitre sa configuration réseau, la commande ifconfig

Pour connaitre votre(vos) propre(s) adresse(s) IPv4 ou v6 et adresse(s) matériel (resp. pour Linux inet adr - adr inet6 - HWaddr) associées à votre(vos) interface(s) réseau, vous pouvez utiliser ipconfig sous Windows, ifconfig sur Linux et les systèmes MAC, ou une commande alternative plus récente ip addr show sous Linux.

Chaque interface est identifiée par le mot clé eth suivi d'un numéro (exemple eth0 pour la première interface). Certaines versions de Linux nomment les interfaces par leur type. Par exemple, comme ci-dessous vous pouvez trouver em1 à la place d'eth1, préfixe identifiant les cartes de type Intel. La sortie répertorie généralement toutes les interfaces actives mais peut être limitée aux interfaces sélectionnées si désiré. La commande ip en particulier est à usage multiple et très puissante.
